# Ommatogammarus_phylogeny

This repository contains data and code for analyses for the paper 'The gems of Lake Baikal: Taxonomic relationships within scavenger amphipods of the genus Ommatogammarus sensu lato revisited with genomic data and DNA barcoding' (to be submitted)

Raw data:
  - FCM data in Fig2 folder
  - COI sequences submitted to NCBI in the Fig5 folder
  - 18S sequences submitted to NCBI in the FigS4 folder
  - assembled mitochondrial genomes and annotation + 45S rDNA loci in the Fig3 folder
